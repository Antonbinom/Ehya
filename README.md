# Ehya

Выпускной проект на курсе Web-start в Glo Academy

## О проекте

Верстка одностраничного интернет магазина книг

Функционал

- Адаптивная, валидная, кросбраузерная верстка
- Слайдеры
- Плавный скролл
- Валидация форм
- Отправка формы на email
- Модальное окно
- Видео

## Стек технологий

- HTML
- CSS
- SASS
- JQuery
- Плагины JQuery
- PHP Mailer
- Pixel perfect
- BEM
